1 Ubuntu 설치 (Raspberry Pi Imager)

PC에 Raspberry Pi Imager를 설치하고 microSD 카드에 Ubuntu를 굽는 과정이에요.

Imager 설치

sudo apt install rpi-imager -y rpi-imager

Imager 설정 순서

항목선택값
OSOther general-purpose OS → Ubuntu → Ubuntu Server 22.04 LTS (64-bit)
StoragemicroSD 카드 선택
⚙️ 고급 설정hostname: raspberrypi / username: ubuntu / password: 직접 설정 / Timezone: Asia/Seoul
반드시 톱니바퀴 ⚙️ 를 눌러서 비밀번호를 미리 설정해야 해요. 안 하면 로그인이 안 될 수 있어요.
라즈베리파이 4는 micro HDMI 포트가 2개인데 안쪽(HDMI 0)에 꽂아야 화면이 나와요.

3 PC → 라즈베리파이 파일 전송 (scp)

두 기기가 같은 네트워크(공유기)에 연결되어 있으면 scp 명령어로 파일을 직접 전송할 수 있어요.

라즈베리파이 IP 확인 (라즈베리파이에서)

hostname -I

SSH 비밀번호 인증 활성화 (라즈베리파이에서)

# Ubuntu 22.04는 기본적으로 비밀번호 로그인이 막혀있음 sudo nano /etc/ssh/sshd_config.d/50-cloud-init.conf # PasswordAuthentication yes 로 변경 또는 추가 sudo systemctl restart ssh

파일 전송 (PC에서)

# -o 옵션으로 비밀번호 인증 강제 적용 scp -o PreferredAuthentications=password -r \ /home/appletea/workspace_onedevice_2/Project_Skincare_Recommendation \ ubuntu@10.10.59.22:~/
파일 목록이 쭉 올라가면서 복사되면 성공.
scp 명령어에서 IP 주소에 대괄호 [ ] 를 쓰면 안 돼요. ubuntu@10.10.59.22:~/ 형식으로 사용.

4 GitHub 연결 (git pull로 업데이트)

파일이 복사될 때 .git 폴더도 같이 넘어가서 git이 이미 설정되어 있어요. SSH 키만 GitHub에 등록하면 앞으로 git pull 한 줄로 업데이트할 수 있어요.

① PC에서 라즈베리파이 SSH 접속

ssh ubuntu@10.10.59.22

② 라즈베리파이에서 SSH 키 생성

ssh-keygen -t ed25519 -C "dongs4739@gmail.com" # 엔터 3번 cat ~/.ssh/id_ed25519.pub # 출력된 내용 복사

③ GitHub에 공개키 등록

github.com → Settings → SSH and GPG keys → New SSH key → 붙여넣기 → Add SSH key

④ git remote 확인

cd ~/Project_Skincare_Recommendation git remote -v # origin git@github.com:Appletea404/Skin-Care_Recommendation.git 이 보이면 OK

⑤ 연결 테스트

git pull
Already up to date. 가 뜨면 git 연결 성공!

앞으로 업데이트 방법

# PC에서 코드 수정 후: git push # 라즈베리파이에서: cd ~/Project_Skincare_Recommendation git pull

5 파이썬 환경 설치

필수 패키지 설치

# Java 포함 (KoNLPy 필수) sudo apt update sudo apt install python3-pip python3-venv default-jdk -y

가상환경 생성 및 패키지 설치

cd ~/Project_Skincare_Recommendation python3 -m venv .venv source .venv/bin/activate pip install flask konlpy scikit-learn scipy pandas numpy gunicorn
설치 중 Daemons using outdated libraries 창이 뜨면 그냥 엔터 누르면 돼요.
다음에 다시 접속했을 때 가상환경 활성화: source ~/.venv/bin/activate

6 웹서버 실행

web_app.py 설정 확인

web_app.py 마지막 줄이 아래처럼 되어 있어야 외부에서 접속 가능해요.
if __name__ == '__main__': app.run(host='0.0.0.0', port=5000, debug=False)

gunicorn으로 실행 (권장)

cd ~/Project_Skincare_Recommendation source .venv/bin/activate gunicorn -w 2 -b 0.0.0.0:5000 web_app:app

브라우저에서 접속

http://10.10.59.22:5000
AI 피부 추천 페이지가 뜨면 성공!
첫 번째 추천 요청은 KoNLPy JVM 초기화로 5~15초 걸려요. 두 번째부터는 빠릅니다.

7 부팅 시 자동 실행 설정

라즈베리파이를 껐다 켤 때마다 자동으로 서버가 실행되게 하는 설정이에요.

systemd 서비스 파일 생성

sudo nano /etc/systemd/system/skincare.service

아래 내용 붙여넣기

[Unit] Description=Skincare Flask App After=network.target [Service] User=ubuntu WorkingDirectory=/home/ubuntu/Project_Skincare_Recommendation ExecStart=/home/ubuntu/Project_Skincare_Recommendation/.venv/bin/gunicorn \ -w 2 -b 0.0.0.0:5000 web_app:app Restart=always [Install] WantedBy=multi-user.target

서비스 등록 및 시작

sudo systemctl enable skincare sudo systemctl start skincare # 상태 확인 sudo systemctl status skincare
명령어설명
sudo systemctl start skincare서버 시작
sudo systemctl stop skincare서버 중지
sudo systemctl restart skincare서버 재시작 (코드 업데이트 후)
sudo systemctl status skincare서버 상태 확인

8 외부 IP에서 접속하기 (포트포워딩)

같은 와이파이/랜에서는 10.10.59.22:5000으로 접속하면 돼요.
집 밖에서도 접속하려면 공유기에서 포트포워딩 설정이 필요해요.

공인 IP 확인 (라즈베리파이에서)

curl ifconfig.me

공유기 포트포워딩 설정

항목값
공유기 관리 페이지192.168.0.1 또는 192.168.1.1
외부 포트5000
내부 IP10.10.59.22 (라즈베리파이 IP)
내부 포트5000
프로토콜TCP

방화벽 설정

sudo ufw allow 5000 sudo ufw enable

외부 접속 주소

http://[공인IP]:5000
가정용 인터넷은 공인 IP가 바뀔 수 있어요. 고정 IP를 신청하거나 DuckDNS 같은 무료 DDNS 서비스를 사용하면 도메인으로 접속할 수 있어요.
